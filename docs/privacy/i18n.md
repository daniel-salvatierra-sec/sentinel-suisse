# Internationalisation (i18n)

**Mandatory languages for Sentinel Suisse:** French, German, Spanish, Portuguese, English.

| Code | Language | Primary audience |
|------|----------|------------------|
| `fr` | Français | Switzerland (Romandie), France |
| `de` | Deutsch | Switzerland (Deutschschweiz), Germany, Austria |
| `es` | Español | Spain, Latin America, expat community |
| `pt` | Português | Portugal, Brazil, lusophone community |
| `en` | English | International users, tech default |

**Default API language:** `fr` (Swiss Romandie pilot).

## Implemented today

- Privacy policies: `docs/privacy/privacy-policy.{fr,de,es,pt,en}.md`
- Legal API: `GET /api/v1/legal/privacy?lang=<code>`
- Constants: `src/sentinel_suisse/i18n/__init__.py`

## Future surfaces (same five languages)

- User-facing UI copy and error messages
- Alert notification templates (email / WhatsApp)
- Public marketing pages and onboarding

When adding any user-visible text, ship all five locales in the same PR or phase.
